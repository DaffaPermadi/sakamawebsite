Website Curhat.id PLN UID Jawa Timur

Website curhat.id (Culture From Hearth) PT. PLN UID Jawa Timur yaitu website untuk mengukur dan meningkatkan pemahaman budaya kerja di kalangan karyawan. Website ini bertujuan untuk memfasilitasi assessment terkait budaya kerja, memastikan seluruh karyawan di berbagai wilayah Jawa Timur dapat berpartisipasi secara efektif. Dengan pendekatan yang interaktif dan mudah diakses, Curhat.id menjadi alat strategis dalam membangun keselarasan budaya kerja di lingkungan PLN.

Teknologi:
Laravel Vite Tailwind CSS MySQL Bootstrap

Website Trucking Vifiera Minova Bersaudara

Platform manajemen trucking yang dirancang untuk mendukung operasional logistik secara efisien. Website ini menyediakan fitur-fitur penting seperti pengelolaan DTP (Delivery to Place) dan DTA (Delivery to Address) untuk memantau pengiriman barang secara real-time. Dengan antarmuka yang intuitif, pengguna dapat mengelola armada truk, jadwal pengiriman, dan pelaporan secara terpusat, memastikan kelancaran dalam operasional sehari-hari.

Laravel Filament MySQL
